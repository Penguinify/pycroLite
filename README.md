# Documentation

## Basic Syntax
Using commas you can separate arguments.

`delay,1`

You can add multiple commands by adding a new line.

`delay,1`    
`press,a`

## Cheat Sheet

`delay,#`    
delays program for # seconds    
`press,#`   
presses down #    
`release,#`            
releases #    
`print,#`     
prints # to console    
`counter,[opt]`        
prints amount of loops
